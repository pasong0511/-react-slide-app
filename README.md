# 📡 react-infiinit-auto-slide-app

## 💁‍♂️ Introducing Project

> -   무한 슬라이드를 구현한 결과물 입니다.
> -   좌우 버튼을 통해서 이동 가능합니다.
> -   하단의 슬라이드 페이지 네이션을 통해 이동 가능합니다.
> -   컴포넌트로 제작하여 어디에든 가져다 쓰기 쉽습니다.
